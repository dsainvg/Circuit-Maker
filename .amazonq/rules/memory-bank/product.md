# Circuit Maker - Product Overview

## Project Purpose
Circuit Maker is a Python-based digital circuit synthesis tool that automatically discovers logic circuits to produce desired outputs from given inputs. The system uses tree-based search algorithms to explore combinations of logic gates and find optimal circuit configurations.

## Key Features
- **Automated Circuit Discovery**: Uses tree-based search to systematically explore circuit solutions
- **Flexible Gate Library**: Supports custom logic gates loaded from configuration files
- **CSV Data Integration**: Imports input/output data from CSV files for circuit specification
- **Comprehensive Logging**: Detailed search process logging with progress tracking
- **Configurable Search Complexity**: Adjustable maximum circuit complexity limits
- **Truth Table Validation**: Verifies circuit outputs against expected results

## Target Users
- **Digital Logic Students**: Learning circuit design and Boolean algebra concepts
- **Hardware Engineers**: Prototyping and validating small digital circuits
- **Educators**: Teaching automated design and search algorithms
- **Researchers**: Exploring circuit synthesis and optimization techniques

## Use Cases
- Educational demonstrations of circuit synthesis
- Small-scale logic circuit prototyping
- Automated generation of combinational logic circuits
- Verification of manual circuit designs
- Research into circuit optimization algorithms
